# Product Manager Skill

A comprehensive product management skill for AI assistants, covering the complete product management knowledge system including strategy, user research, requirements analysis, roadmap planning, data-driven decision making, and more.

## Overview

This skill transforms your AI assistant into an experienced professional product manager with keen market insight and user empathy, capable of driving products from concept to market launch.

## Features

- ✅ **Product Strategy & Vision** - Market opportunities, business models, positioning
- ✅ **User Research** - Personas, interviews, surveys, usability testing, JTBD
- ✅ **Requirements Management** - Collection, validation, prioritization, PRDs
- ✅ **Product Planning** - Roadmaps, release planning, iteration management
- ✅ **Prioritization** - MoSCoW, RICE scoring, Kano model
- ✅ **Data-Driven Decisions** - Metrics, analytics, A/B testing
- ✅ **Competitive Analysis** - SWOT, feature comparison, market positioning
- ✅ **Product Launch** - GTM strategy, launch planning, monitoring
- ✅ **Growth & Optimization** - AARRR funnel, retention, conversion

## Installation

### For Claude Code
```bash
# Clone to your Claude Code skills directory
git clone https://github.com/xdmjun/product-manager.git ~/.claude/skills/product-manager
```

### For Other Tools (Cursor, Aider, Windsurf, etc.)
Copy the `SKILL.md` file to your tool's skills directory.

## Usage

Once installed, you can activate the skill by saying:

```
"Activate Product Manager mode and help me with competitive analysis"
```

Or:

```
"As a product manager, create a product roadmap for my SaaS product"
```
